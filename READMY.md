# Integrity Shield Pro (Makhniei)

**Integrity Shield Pro** — це програмний комплекс для захисту даних від несанкціонованої модифікації та атак типу Ransomware (вірусів-шифрувальників).

## Основні можливості
* **Моніторинг у реальному часі:** Використання драйверів ядра Windows для відстеження змін файлової системи.
* **Захист від Ransomware:** Аналіз ентропії файлів та сигнатур для виявлення шифрування.
* **Lock & Hold:** Унікальний алгоритм блокування файлу при спробі атаки, що не дає вірусу перезаписати дані.
* **Автоматичне відновлення:** Миттєве відновлення пошкоджених файлів із захищених тіньових копій.
* **GPS-трекер файлів:** Система не втрачає файл, навіть якщо його перемістити в іншу папку.
* **Сповіщення:** Відправка звітів про інциденти на Email.

## Технології
* **Python 3.12+**
* **CustomTkinter** (Сучасний GUI)
* **Watchdog** (Моніторинг подій)
* **Cryptography (Fernet/AES)** (Шифрування бази даних хешів)
* **SQLite (WAL Mode)** (Швидке зберігання метаданих)
* **PyStray** (Робота в системному треї)
